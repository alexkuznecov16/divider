Kā tas darbojas?

1. Ievads

Šis projekts tika izveidots ar mērķi ļaut lietotājiem atrast lielāko kopīgo dalītāju. Zemāk redzams galvenais JavaScript kods, kas ieviests šajā projektā

2. JavaScript kods

Projektā tiek izmantotas divas funkcijas:
  1. Pārvaldnieka funkcija 'mainFunction()' datu iegūšanai no lietotāja, visu pārējo funkciju izsaukšanai un gala rezultātu izdrukāšanai (šī funkcija tiek aktivizēta, noklikšķinot uz pogas 'Solve');
  2. funkcija vērtību testēšanai un rezultatu atrisinasanai 'checkFunction(skaitlis1, skaitlis2)', lietotaja ievaditi 2 skaitli, pec kuram ir jaatrast lielako kopigo dalitaju;
Šī projekta galvenās funkcijas JavaScript kods ir parādīts laukā zemāk: 
 
while (x2 != 0) {
  y1 = x2; // make new variable for x2 saving
  x2 = x1 % x2; // divide and get remainder
  x1 = y1; // refresh x1 for next work in loop
}
// kad x2 driz bus 0 tad rezultats bus nosutits no x1 mainigaja
Noklikšķiniet uz šīs ikonas , lai skatītu šī JavaScript koda blokshēmu.
